2. 把第n个盘子放到C塔上
3. 将n-1个位于辅助塔B盘子放到C塔上
```Python
def hanoi(n, a, b, c):
	if n > 0:
		hanoi(n - 1, a, c, b)
		print("moving from %s to %s" % (a, c))
		hanoi(n - 1, b, c, a)
```
