> [!note] 从列表中查找指定元素
> - 输入: 列表, 待查找元素
> - 输出: 元素下标(未找到一般返回None或-1)

# 1. 顺序/线性查找
无序或有序都行
```Python
def linear_search(lis, val):
    for ind, v in enumerate(lis):
        if v == val:
            return ind
    else:
        return -1
```

# 2. 二分查找/折半查找
要求必须有序
从中间找, 对比查找的值和候选区的中间值, 使候选区减少一半
```python
def binary_search(lis, val): 
	left = 0 
	right = len(lis) - 1 
	while left <= right: # 候选区有值 
		mid = (left + right) // 2 
		if lis[mid] == val: 
			return mid 
		elif lis[mid] > val: 
			right = mid - 1 
		else: left = mid + 1 
	else: return None
```
